# VocalRise — Tavish's Virtual Vocal Coach

## What It Is
VocalRise is a singing practice app purpose-built for a complete beginner learning to sing for musical theatre. Its standout feature is real-time pitch detection: it listens to the user sing assigned song parts, identifies wrong notes, and explains in plain language how to fix them — functioning like a patient, always-available virtual vocal coach.

## Primary User
Tavish — a complete singing beginner (no prior choir, lessons, or performance experience) who is:
- Enrolled in a musical theatre course with assigned song parts (ensemble or solo) accompanied by sheet music or model vocal audio
- Working toward a first audition for a musical (possibly on a 2-week timeline)
- An occasional comedic/improv performer who wants to add singing to their live performance skillset

## Core Feature: Pitch Detection & Correction Coaching
The heart of the app. The user selects a song part to practice, the app listens via microphone as they sing, detects each note in real time, highlights any wrong notes, and delivers specific, beginner-friendly coaching on how to fix them (e.g. "That note was a bit flat — try lifting the back of your tongue slightly"). Feels like a real vocal coach catching mistakes and guiding correction.

## Core Use Cases
1. **Course assignment practice** — Learn and rehearse specific assigned song parts using sheet music or a model audio reference track
2. **Audition preparation** — Structured preparation of a 1-minute solo musical theatre song on a tight timeline
3. **Vocal foundations** — Build core technique: breathing, posture, pitch, tone, and range — from scratch
4. **Musical improv readiness** — Develop spontaneous singing comfort for comedic/improv live performances

## Key Features
- **Real-Time Pitch Detection** — Microphone listens as user sings; live pitch accuracy overlay; wrong notes flagged immediately
- **Plain-Language Correction Tips** — When a note is wrong, the app explains what went wrong and how to fix it in beginner-friendly terms
- **Guided Vocal Lessons** — Short daily modules from breathing basics to musical theatre technique
- **Daily Vocal Warm-Ups** — Adaptive scales, lip trills, and arpeggios matched to vocal range
- **Musical Theatre Song Practice** — Scrolling lyrics, pitch overlay, model audio reference, and a performance score
- **Audition Prep Mode** — Set target song and audition date; receive a focused daily practice plan with a readiness score
- **Progress Tracking & Streaks** — Pitch trend dashboards, vocal range growth, daily consistency streaks
